Bar plot

Este tipo de grfica nos permite mostrar variables categoricas o que contienen texto, lo que en la ciencia de datos es importante para la visualizacin de datos.
Matplotlib nos ofrece ciertas funciones para este tipo de grficas

In[]:
import matplotlib.pyplot as plt
import numpy as np
In[]:
#Definiendo variables
paises = ['In','Ja','Mx','Col','Ale']
poblacion = [1000,800,500,700,285]
In[]:
#Haciendo grficas de barras
plt.bar(paises,poblacion)
Out[]:
<BarContainer object of 5 artists>
In[]:
#Personalizando
#Haciendo grficas de barras
plt.bar(paises,poblacion,width=0.5, color='red')
Out[]:
<BarContainer object of 5 artists>
In[]:
#Alternando colores
#Haciendo grficas de barras
plt.bar(paises,poblacion,width=0.5, color=['red','blue'])
Out[]:
<BarContainer object of 5 artists>

Modificando parametros:

Supongamos que no puedo acceder a las variables categoricas y simplemente entro a una de las propiedades:

  • plt.xticks: Configuraciones del eje x
    • np.arange(5): que me de una lista o posiciones del arreglo y as poder entrar al indice del eje x
In[]:
#Alternando colores
#Haciendo grficas de barras
#Cambiando configuraciones de eje x
plt.xticks(np.arange(5),('India','Japn','Mexico','Colombia','Alemania'))
plt.bar(paises,poblacion,width=0.5, color=['red','blue','c','yellow','gray'])
Out[]:
<BarContainer object of 5 artists>

Rotando para evitar el solapamiento

In[]:
#Alternando colores
#Cambiando configuraciones de eje x
plt.xticks(np.arange(5),('India','Japn','Mexico','Colombia','Alemania'),rotation=45)
plt.bar(paises,poblacion,width=0.5, color=['red','blue','c','green','gray'])
Out[]:
<BarContainer object of 5 artists>

Graficando barras horizontales

In[]:

plt.barh(paises,poblacion,height=0.5, color=['red','blue','c','green','gray'])
