import matplotlib.pyplot as plt
import numpy as np
In[]:
data = np.random.randint(1,50,100)

Histograma

In[]:
#Analizar distribucin y frecuencia
#histograma
plt.hist(data)
#bins = 10 por default
plt.show()

Bins = Nmero de barras en la cul va a llegar la distribucin de datos

In[]:
#Jugar cn una variable importante 
#Bins en histogramas
plt.hist(data,bins=50)
plt.show()
In[]:
#Modificando parametros
plt.hist(data,bins=15,histtype='bar')
plt.show()
In[]:
#Modificando parametros
plt.hist(data,bins=15,histtype='step')
plt.show()

Boxplot

Grafico de cajas y bigotes

In[]:
#Boxplot
plt.boxplot(data)
plt.show()
In[]:
#Explorando propiedades
plt.boxplot(data,vert=False)
plt.show()
In[]:
#Explorando propiedades
plt.boxplot(data,vert=False,patch_artist=True,notch=True)
#Notch me dice como se encuentran los datos en distribuacin 
plt.show()
In[]:
#Agregando data de tipo outlayer (con anomalias)
data=np.append(data,200)
#Explorando propiedades
plt.boxplot(data,vert=False,patch_artist=True,notch=True)
#Notch me dice como se encuentran los datos en distribuacin 
plt.show()

Aqu toma en cuenta el data anomalo. Podemos quitarlo con la sentencia:
showfliers=False

In[]:
#Agregando data de tipo outlayer (con anomalias)

#Explorando propiedades
plt.boxplot(data,vert=False,patch_artist=True,notch=True,showfliers=False)

plt.show()

Si le quito el show out layers

Me muestra todos los datos con los que estoy trabajando.
showfliers=True

Scatterplot (Dispersin)

In[]:
N = 50
x = np.random.rand(N)
y = np.random.rand(N)
area = (30*np.random.rand(N))**2
colors = np.random.rand(N)
In[]:
#Graficando Scatter ()
plt.scatter(x,y)
Out[]:
<matplotlib.collections.PathCollection at 0x7f88e5721ca0>
In[]:
#Graficando Scatter ()
#Agregando ms variable
plt.scatter(x,y,s=area)
Out[]:
<matplotlib.collections.PathCollection at 0x7f88e59bad50>

Explicacin:

El tamao de las burbujas va a tener referencia a una 3ra variable que es: area Qu pasa si agrego otra variable?

In[]:
#Graficando Scatter ()
#Agregando ms variable
plt.scatter(x,y,s=area,c=colors)
Out[]:
<matplotlib.collections.PathCollection at 0x7f88e61a7200>

Con esto de los colores me podra servir para hacer una clasificacin de lo que estoy haciendo.

In[]:
#Cambiando markers
#Graficando Scatter ()
plt.scatter(x,y,s=area,c=colors,marker='o')
Out[]:

Ajustando Alpha

In[]:
#Cambiando markers
#Graficando Scatter ()
plt.scatter(x,y,s=area,c=colors,marker='o',alpha=0.5)
plt.show()

De esta manera puedo ver si hay puntos o datos sobrepuestos
